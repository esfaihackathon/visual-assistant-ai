The Problem

Banking apps exclude hundreds of millions of users

👁️

Visual Barriers

80M+ visually impaired Indians cannot read standard app UI. Small fonts, low contrast, and colour-coded status are completely inaccessible.

🤲

Motor Barriers

300M+ senior citizens struggle with precise tapping on small buttons. Biometric authentication requires fine-motor button presses.

📖

Literacy Barriers

400M+ rural users with limited literacy cannot navigate menu-based interfaces or read account numbers and transaction details.

🎨

Colour-Blindness

300M+ colour-blind users cannot distinguish red/green status indicators used across virtually every banking app.

"I have to call my son every time I want to check my balance." — Typical elderly banking customer

Our Solution

Saral — Voice-First Banking

A banking assistant that works entirely by voice. The visual UI is a high-contrast fallback — not the primary interface.

🎙️

Voice is Primary

Every banking action — balance, transfer, cheque book, transactions — is completable by voice alone.

👆

Touch is Secondary

88dp minimum tap targets. Large-font cards. Beneficiaries shown as full-width tappable rows.

♿

WCAG AAA

15.8:1 contrast ratio. Colour-blind safe palette. Full ARIA labels. Screen reader compatible.

🔒

Auto Fingerprint Auth

When a transfer is confirmed, fingerprint scanner activates automatically — no button tap required.

📍

Fixed Mic Button

The microphone button is anchored at the bottom of every screen and never scrolls away — always within reach.

💬

Conversational Follow-ups

After every action the app asks what you'd like to do next — no need to remember commands.

Technical Architecture

How we built it

Android App

Kotlin + Jetpack Compose ├── MVVM + Hilt DI ├── StateFlow / SharedFlow ├── BiometricPrompt (auto-trigger) ├── SpeechRecognizer API ├── TextToSpeech API └── Compose fixed bottom mic bar // LaunchedEffect(step) fires biometric // when step == AwaitingBiometric

Web App

HTML + CSS + Vanilla JS ├── Web Speech API (voice in/out) ├── State machine │ currentTransferPhase │ awaitingSimpleFollowUp │ awaitingPostTransactionChoice ├── CSS flex layout │ home-scroll-content (flex:1) │ mic-bar (flex-shrink:0) ← fixed └── Auto biometric // setTimeout(executeWebTransfer, 2200) // after entering biometric phase
